PySymEmu
========
* An intel 64 symbolic basic block analysis model. 
* ``cpu.py`` is from https://github.com/feliam/pysymemu

Dependencies:
-------------
* z3, an smt solver. http://z3.codeplex.com/ 
* distorm3, a disassembler. https://code.google.com/p/distorm/

How to Use:
------------
* modify 1.py and run
